## Pourquoi Statify ?

**Vous utilisez Google Analytics ?** Vos données de visiteurs sont envoyées et stockées sur les serveurs de Google aux États-Unis. Avec Matomo, l'installation est complexe et les ressources consommées sont importantes. Statify prend 2 minutes à installer et vos données ne quittent jamais votre hébergement.

|  | Google Analytics | Matomo | **Statify** |
|---|:---:|:---:|:---:|
| Données stockées sur votre hébergement | ❌ | ✅ | ✅ |
| Stats d'engagement *(durée, pages vues, sessions)* | ✅ | ✅ | ✅ |
| Géolocalisation des visiteurs | ✅ | ✅ | ✅ |
| Gratuit & open source | ❌ | ✅ | ✅ |
| Script léger *(< 5 Ko, ne ralentit pas votre site)* | ❌ | ❌ | ✅ |
| Installation simple *(pas de config serveur)* | ✅ | ❌ | ✅ |
| Conforme RGPD sans configuration complexe | ❌ | ❌ | ✅ |
| Zéro abonnement mensuel | ❌ | ❌ | ✅ |
